# FinancialResearch Frontend

Future frontend stack: React + Vite.

Principles:

- The frontend calls only the FinancialResearch backend.
- The frontend must not call Parse Lab directly.
- Parse Lab remains an independent backend service behind FinancialResearch.
- No npm dependencies are installed in this skeleton phase.

Planned pages:

- Dashboard
- Company Research
- Financial Report Analysis
- Macro Environment
- Sentiment Monitoring
- Market Data Tracking
- Research Repository
- Decision Support
- System Tasks
